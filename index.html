Registration-Form

In summary, this script aims to enhance the end-user experience by providing a visual feedback in form of validation. Error, or success messages will be displayed dynamically, guiding an end-user through some input requirements. In case the validation process will get passed, the end-user will be redirected to the next page, incorporating the username as a parameter. This script uses browser's local storage by storing user data temporarily. It won't just retrieve the existing data, it will also put the current used form of data together, and store it back.
